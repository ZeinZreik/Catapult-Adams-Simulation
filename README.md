# Catapult Adams Simulation

A mechanical simulation project focused on modeling and evaluating a catapult mechanism in MSC Adams. The repository presents a practical dynamics workflow for mechanism design, motion analysis, and performance review.

## Project Snapshot

- Domain: multibody dynamics, mechanical design, simulation engineering
- Main tool: MSC Adams
- Focus: catapult linkage behavior, motion response, and mechanical performance
- Portfolio value: demonstrates simulation-led design thinking for mechatronics and robotics systems

## What This Project Demonstrates

- Building and validating a mechanism model before hardware prototyping
- Studying motion, forces, constraints, and dynamic behavior in a virtual environment
- Translating a physical mechanism concept into an analyzable engineering model
- Presenting simulation work as part of a broader robotics and mechatronics portfolio

## Technical Focus

This project is useful for showing experience with:

- Multibody simulation
- Mechanical system modeling
- Kinematic and dynamic analysis
- CAD-to-simulation workflows
- Mechatronics design validation
